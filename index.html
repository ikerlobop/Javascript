Ejemplo: Cliente Modbus TCP con Python

from pymodbus.client.tcp import ModbusTcpClient # Crear cliente client = ModbusTcpClient("192.168.2.221", port=502) # Conectar client.connect() # Leer holding registers response = client.read_holding_registers(address=0, count=12, slave=1) # Verificar si la respuesta es válida if response and not response.isError(): print(f"Holding registers: {response.registers}") else: print("Error al leer los holding registers") # Cerrar conexión client.close()

Explicación del código

  • ModbusTcpClient: Se importa del módulo pymodbus para crear un cliente TCP.
  • client.connect(): Establece conexión con el servidor Modbus.
  • read_holding_registers(...): Solicita leer 12 registros a partir de la dirección 0 del esclavo con ID 1.
  • response.isError(): Verifica si ocurrió un error en la lectura.
  • print(...): Muestra los registros leídos si la respuesta fue exitosa.
  • client.close(): Cierra la conexión Modbus.
